# Nik

An AI life assistant for iOS + Android. Habits, focus, family circle, health, money, diary, voice + AI chat.

Built with **React 19 + TypeScript + Vite**, wrapped in **Capacitor 8** for native iOS/Android. Visual design carried over from the original Claude-generated prototype with full fidelity (5 theme universes, oklch color system, glass tiles, gradient bento layouts, framer-motion animations).

## Quick start

```bash
# Install web deps
cd web
npm install

# Dev (browser, HMR)
npm run dev
# → http://localhost:5173

# Build for native
npm run build      # → ../dist/

# Run on iOS simulator (requires Xcode + macOS)
cd ..
npx cap sync ios
npx cap run ios
```

## Stack

- **Frontend**: React 19, TypeScript 5.9, Vite 8, framer-motion 12
- **Native shell**: Capacitor 8 (iOS native, Android coming)
- **Theming**: CSS custom properties + `data-theme` attribute (5 universes)
- **Backend** (planned): Supabase (Postgres + Auth + Realtime + Storage), with the path to either Supabase Cloud or Google Cloud (Cloud Run + Cloud SQL)
- **AI**: Anthropic Claude API + MCP servers (planned)

## Status

Frontend is fully functional with mock data. All 22 screens render with their bento layouts, theme switching, and animations. Tab navigation, screen state, and the "More" catalog (55 dashboards across 9 categories with a premium "Coming Soon" preview for unimplemented items) all work.

Backend wiring (Supabase) is the next phase. See [`docs/Backlog.md`](docs/Backlog.md) for the roadmap.
