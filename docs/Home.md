## Quick orientation

**What this is:** A design-led iOS + Android app. AI life assistant — habits, focus, family, health, money, diary, voice/AI chat. Built from a Claude-generated React DOM prototype, hardened into a real Vite + TypeScript app, wrapped with Capacitor for native shipping.

**Why these tools:** See [[Decisions/001 — Capacitor over React Native]] and [[Decisions/002 — Vite over Babel-in-browser]].

**Where the source lives:**
- Production app: [`web/src/`](../web/src/) (Vite + React 19 + TS)
- Native shell: [`ios/`](../ios/) (auto-generated by Capacitor)
- Original prototype: [`www/`](../www/) (kept for reference, no longer the source of truth)

**Where to start coding:** [[Build & Run]] → install + dev server in 30 seconds.
